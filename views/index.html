Image Search Abstraction Layer

How It Works

Visit /search/(yoursearchtermhere) to get a list of the first ten results via the Flickr API. Add the query string ?offset(number) to get results further back from your search, e.g. ?offset=2 to get the next ten, and so on.

  • https://imgsearch-jsanderson.glitch.me/search/puppies
  • https://imgsearch-jsanderson.glitch.me/search/puppies?offset=2

Visit /recent to get the ten most recently used search terms, along with the date made.

  • https://imgsearch-jsanderson.glitch.me/recent
